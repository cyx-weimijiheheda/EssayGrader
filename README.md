# 《高中英语应用文批量批改工具》

> 一个 PySide6 桌面应用。把一叠手写作文照片拖进去，它会认字、改错、打分、写评语，再给你一篇更漂亮的范文。

---

## 一、它能做什么

| 步骤 | 干了什么 | 用什么干的 |
|---|---|---|
| 扫码 | 扫描作文图片上的条形码，识别考号 | pyzbar |
| OCR | 把手写英文变成文本，顺便认姓名和班级 | 阿里云 Qwen-VL / 本地 RapidOCR |
| 校字 | 把 OCR 认错的地方修掉（但绝不碰学生自己的语法错误） | DeepSeek |
| 打分 | 按高考标准评分：内容5分 + 语言5分 + 结构3分 + 格式2分 = 15分 | DeepSeek |
| 改错 | 逐处标出错误，用 `[错误:原文→修改|理由]` 格式嵌在修正版里 | DeepSeek |
| 写评语 | 100-200字中文评语，点亮点、给建议 | DeepSeek |
| 升格 | 基于学生原文，写一篇同样意思但更漂亮的范文 | DeepSeek |
| 导出 | 生成排好版的 Word 文档，每人一页 | python-docx |

---

## 二、怎么用

### 安装

```bash
pip install -r requirements.txt
```

依赖：PySide6、requests、rapidocr-onnxruntime、pyzbar、Pillow。

如果用阿里云 Qwen 做 OCR（推荐），除此之外无需其他配置。如果用本地 RapidOCR（离线模式），什么 API Key 都不用填。

### 配置

打开软件 → **文件 → 设置**，填入：

- **DeepSeek API Key**：必填，批改评分全靠它
- **Qwen API Key**：用 Qwen OCR 时填，用 RapidOCR 可跳过
- **OCR 方式**：Qwen（推荐，手写识别更准）或 RapidOCR（离线，无需联网）

其余选项——是否打分、是否写评语、是否生成改错修正版、是否升格——用复选框控制，按需开关。

### 批改

1. 在顶部输入框粘贴**作文题目**
2. 点击 **浏览文件夹**，选择装满作文照片的文件夹（支持 jpg/jpeg/png/bmp）
3. 点 **开始批改**
4. 等进度条走完。每篇作文会在左侧列表出现，点击即可查看详情
5. 完成后导出 **JSON** 或 **DOCX**

### 去重

同一个学生的同一张照片不会被重复批改。程序用 SHA256 比对文件内容，按考号 > 姓名 > 文件名匹配历史记录。关掉软件再开也不会丢——缓存存在 `graded_cache.json` 里。

### 附加要求

在设置里可以写自定义指令，例如：

> 重点关注时态错误；本次作文为建议信，请重点检查建议句型的多样性

这个指令会被注入到批改和 OCR 修正的每个阶段。

---

## 三、导出格式

**JSON** 是完整的原始数据，方便二次处理。

**DOCX** 是排好版的 Word 文档：
- 每位学生一个独立节，页眉带姓名和考号
- 评分表带彩色表头
- 修正版用红色删除线标错误、绿色标修改建议
- 升格范文带蓝色左边框
- 如果学生严重偏题，还会额外生成一篇"学习版范文"，用黄色左边框

## 四、文件说明

| 文件 | 职责 |
|---|---|
| `essay_grader.py` | 主程序，GUI 界面 + 批改线程 |
| `prompts.py` | 全部提示词模板（OCR修正 / 批改评分 / 升格范文） |
| `settings_dialog.py` | 设置对话框 + 配置文件读写 |
| `export_docx.py` | DOCX 导出模块 |
| `essay_grader.spec` | PyInstaller 打包脚本 |
| `essay_grader_config.json` | 运行时配置（含 API Key，已 gitignore） |
| `graded_cache.json` | 已批改记录缓存（已 gitignore） |

---

## 五、打包成 EXE

```bash
pyinstaller essay_grader.spec
```

会在 `dist/` 下生成 `essay_grader.exe`。如果分发出去用，可以把 `essay_grader_config.json` 删掉，每个用户第一次打开时在设置里填自己的 API Key。

## 六、设计原则

> 如果你不确定某个错误是 OCR 造成的还是学生本身写错的，优先保留原样。宁可漏过 OCR 错误，也不能误改学生原文。

这个工具的每一个提示词都围绕一个原则：**机器辅助人，但不替代人**。OCR 修正只修识别错误、不碰学生原文；批改评语要求具体、可操作；升格范文保留学生的核心内容和观点，只是帮孩子把话说得更漂亮。

打分标准是高考的，但它的目的不是冰冷地排名——是让一个疲惫的老师在批改两个班之后，还能确保每个孩子都收到一句有用的评语。
